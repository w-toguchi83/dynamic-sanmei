# 第8章　合・冲・刑・害（ごう・ちゅう・けい・がい）


## 8.1 十干の合（じっかんのごう）

十干同士には特定の「合」の関係があり、合すると五行が変化します（化合）。命式内や運との間で起きる場合に重要な変化をもたらします。


| 合 | 変化後の五行 | 象意 |
| --- | --- | --- |
| 甲・己合 | 土 | 正財合・安定・現実志向の結合 |
| 乙・庚合 | 金 | 仁義合・義理・律義さの結合 |
| 丙・辛合 | 水 | 威制合・権威・厳格な結合 |
| 丁・壬合 | 木 | 淫乱合・感情・情緒的な結合 |
| 戊・癸合 | 火 | 無情合・クールな情の結合 |


## 8.2 十二支の合・冲・刑・害


### 8.2.1 六合（りくごう）

十二支のうち特定の二支が「合」することで、その五行が強まります。


| 六合の組み合わせ | 変化 |
| --- | --- |
| 子・丑合 | 土 |
| 寅・亥合 | 木 |
| 卯・戌合 | 火 |
| 辰・酉合 | 金 |
| 巳・申合 | 水 |
| 午・未合 | 火・土 |


### 8.2.2 三合局（さんごうきょく）

三つの地支が組み合わさることで特定の五行が極めて強力になる「三合局」があります。


| 三合局 | 構成支 | 五行 |
| --- | --- | --- |
| 木局 | 亥・卯・未 | 木 |
| 火局 | 寅・午・戌 | 火 |
| 土局 | 巳・酉・丑 | 金（一説では土） |
| 水局 | 申・子・辰 | 水 |


### 8.2.3 六冲（りくちゅう）

正反対の関係にある地支同士が「冲（ぶつかる）」関係で、変動・分離・移動を意味します。


| 六冲の組み合わせ |
| --- |
| 子⟺午 |
| 丑⟺未 |
| 寅⟺申 |
| 卯⟺酉 |
| 辰⟺戌 |
| 巳⟺亥 |


### 8.2.4 刑（けい）と害（がい）

刑は「傷つけ合う」関係、害は「害する」関係で、どちらも困難・トラブルの暗示です。


| 三刑（さんけい） | 寅・巳・申 / 丑・戌・未（相互に刑） |
| --- | --- |
| 自刑（じけい） | 辰・午・酉・亥（自分自身を刑する） |
| 六害（りくがい） | 子⟺未、丑⟺午、寅⟺巳、卯⟺辰、申⟺亥、酉⟺戌 |
