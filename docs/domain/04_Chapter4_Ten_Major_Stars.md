# 第4章　十大主星（じゅうだいしゅせい）


## 4.1 十大主星の概要

十大主星は日干と各干（蔵干を含む）の五行関係から算出される10種類の星で、その人の先天的な気質・能力・行動パターンを示します。「陽占（ようせん）」の核心的な要素です。

💡 十大主星は「日干から見た他の干の五行的関係」で決まります。相生・相剋・比和の方向と陰陽の組み合わせで10種が生まれます。


| 星名 | 日干との関係 | 五行方向 | 陰陽 | 象意・キーワード |
| --- | --- | --- | --- | --- |
| 貫索星（かんさくせい） | 比劫 | 同五行・陽比 | 陽 | 自我・独立・頑固・一徹・自立心 |
| 石門星（せきもんせい） | 比劫 | 同五行・陰比 | 陰 | 協調・和合・社交・平和主義・集団行動 |
| 鳳閣星（ほうかくせい） | 食傷 | 日干が生む・陽 | 陽 | 自由・創造・表現・奉仕・芸術 |
| 調舒星（ちょうじょせい） | 食傷 | 日干が生む・陰 | 陰 | 感受性・芸術・こだわり・完璧主義 |
| 禄存星（ろくぞんせい） | 財星 | 日干が剋す・陽 | 陽 | 現実・財・奉仕・世話好き・責任感 |
| 司禄星（しろくせい） | 財星 | 日干が剋す・陰 | 陰 | 蓄財・堅実・節約・管理・家庭的 |
| 車騎星（しゃきせい） | 官星 | 日干を剋す・陽 | 陽 | 行動力・闘争心・積極性・スポーツ |
| 牽牛星（けんぎゅうせい） | 官星 | 日干を剋す・陰 | 陰 | 誇り・完璧主義・責任・プロ意識 |
| 龍高星（りゅうこうせい） | 印綬 | 日干を生む・陽 | 陽 | 学習・変化・自由・多才・海外志向 |
| 玉堂星（ぎょくどうせい） | 印綬 | 日干を生む・陰 | 陰 | 知識・保守・家系・伝統・堅実 |


## 4.2 十大主星の配置と意味


### 4.2.1 三才（さんさい）への配置

命式の三つの位置（天・人・地）に星が置かれます。どの位置にどの星があるかで意味が変わります。


| 天（年柱天干） | 先祖・外部からの影響・社会的な顔・20代頃 |
| --- | --- |
| 人（月柱天干） | 両親・職業・社会的立場・30〜40代頃 |
| 地（日柱天干）＝日干 | 自分自身（日干そのものが「地」の主） |
| 日支（日柱地支の蔵干） | 配偶者星・パートナーシップ |


### 4.2.2 中殺（ちゅうさつ）との関係

後述する「宿命中殺」により、特定の位置にある星が「中殺」状態になると、その星の持つ能力・宿命が「消える・変容する」という重要な概念です。


## 4.3 十大主星の算出ロジック

日干（甲〜癸）と対象の干（甲〜癸）の関係から、以下のルールで星を特定します。


| 関係 | 対象の五行 | 陽陰 | → 星 |
| --- | --- | --- | --- |
| 比劫（同五行） | 同じ五行 | 同陰陽（同士） | 貫索星 |
| 比劫（同五行） | 同じ五行 | 異陰陽（違い） | 石門星 |
| 食傷（日干が生む） | 日干が生む五行 | 日干と同陰陽 | 鳳閣星 |
| 食傷（日干が生む） | 日干が生む五行 | 日干と異陰陽 | 調舒星 |
| 財星（日干が剋す） | 日干が剋す五行 | 日干と同陰陽 | 禄存星 |
| 財星（日干が剋す） | 日干が剋す五行 | 日干と異陰陽 | 司禄星 |
| 官星（日干を剋す） | 日干を剋す五行 | 日干と異陰陽 | 車騎星 |
| 官星（日干を剋す） | 日干を剋す五行 | 日干と同陰陽 | 牽牛星 |
| 印綬（日干を生む） | 日干を生む五行 | 日干と異陰陽 | 龍高星 |
| 印綬（日干を生む） | 日干を生む五行 | 日干と同陰陽 | 玉堂星 |

⚠ 陰陽の「同・異」の組み合わせルールは流派によって微妙に異なります。実装前に採用する体系を明確に定義してください。
