# 第5章　十二大従星（じゅうにだいじゅうせい）


## 5.1 十二大従星の概要

十二大従星は日干と各地支（年支・月支・日支）の関係から算出される12種類の星で、その人の「生命力の状態・人生の行動パターン・エネルギーの質」を示します。肉体的・物質的な側面を表すとも言われます。


| 星名 | 読み | 生命力 | 主な象意 |
| --- | --- | --- | --- |
| 天報星 | てんぽうせい | 胎（たい） | 種・潜在・神秘・霊的・始まりの前段階 |
| 天印星 | てんいんせい | 養（よう） | 育まれる・受け身・純粋・幼少期的感性 |
| 天貴星 | てんきせい | 長生（ちょうせい） | 生まれたて・可能性・才能・フレッシュ |
| 天恍星 | てんこうせい | 沐浴（もくよく） | 変化・破壊と再生・色情・波乱・魅力 |
| 天南星 | てんなんせい | 冠帯（かんたい） | 社会的活躍・華やかさ・自信・虚栄心 |
| 天禄星 | てんろくせい | 建禄（けんろく） | 安定・自立・実力・堅実・独立 |
| 天将星 | てんしょうせい | 帝旺（ていおう） | 最強・カリスマ・支配・孤高・頂点 |
| 天堂星 | てんどうせい | 衰（すい） | 円熟・指導者・経験・落ち着き・晩年的 |
| 天胡星 | てんこせい | 病（びょう） | 繊細・芸術・アンテナ豊か・不安定 |
| 天極星 | てんきょくせい | 死（し） | 集中・研究・専門・静的エネルギー |
| 天庫星 | てんこせい | 墓（ぼ） | 蓄積・保管・堅実・過去・骨董 |
| 天馳星 | てんちせい | 絶（ぜつ） | 変化・移動・分離・奔放・自由 |


## 5.2 十二大従星の算出

十二大従星は「日干の五行と各地支の関係」から決まります。具体的には、日干の五行を「帝旺（最強の状態）」とした時の各十二支の「十二運（じゅうにうん）」のポジションを参照します。


### 算出の基本手順

- 日干の五行（木・火・土・金・水）を特定する
- その五行の「帝旺支（ていおうし）」を確認する
- 対象の地支が帝旺支から何番目かを数え、十二運表で対応する従星を求める

| 日干の五行 | 帝旺支（最強の地支） |
| --- | --- |
| 木（甲・乙） | 卯（う） |
| 火（丙・丁） | 午（うま） |
| 土（戊・己） | 戌（いぬ）※または未 |
| 金（庚・辛） | 酉（とり） |
| 水（壬・癸） | 子（ね） |

⚠ 土の五行（戊・己）の帝旺支については、戊は戌・己は午とする説など流派による違いがあります。
