# 第6章　中殺（ちゅうさつ）


## 6.1 天中殺（てんちゅうさつ）の基礎

天中殺とは、六十干支の干と支の対応関係において、十干（10個）と十二支（12個）の差から生じる「余り」の2支が天中殺支となる概念です。この2支の年・月・日が「天中殺の時期」となります。


### 6.1.1 天中殺の種類（六種類）


| 種類 | 中殺となる十二支 | 対応する生まれ年（干支の下2桁） | 特徴・象意 |
| --- | --- | --- | --- |
| 子丑天中殺 | 子・丑 | 寅〜亥年生まれ（甲寅〜癸亥） | 晩年型・慎重・蓄積型・晩成 |
| 寅卯天中殺 | 寅・卯 | 辰〜丑年生まれ（甲辰〜癸丑） | 変化型・行動力・突破力 |
| 辰巳天中殺 | 辰・巳 | 午〜卯年生まれ（甲午〜癸卯） | 感情型・芸術・波乱 |
| 午未天中殺 | 午・未 | 申〜巳年生まれ（甲申〜癸巳） | 社会型・能力・人気 |
| 申酉天中殺 | 申・酉 | 戌〜未年生まれ（甲戌〜癸未） | 精神型・研究・専門 |
| 戌亥天中殺 | 戌・亥 | 子〜酉年生まれ（甲子〜癸酉） | 現実型・実行力・目立つ |


## 6.2 宿命中殺（しゅくめいちゅうさつ）

宿命中殺は算命学独自の重要概念で、命式の中の特定の干支が中殺に当たる場合を指します。天中殺より強い影響を持ち、その人の宿命的な課題・才能・人生のテーマに関わります。


### 6.2.1 宿命中殺の種類


| 種類 | 中殺が当たる位置 | 象意・人生のテーマ |
| --- | --- | --- |
| 年支中殺 | 年柱の地支 | 先祖・家系からの解放。社会の枠を超えた生き方 |
| 月支中殺 | 月柱の地支 | 親・家庭からの解放。社会的役割の変容 |
| 日支中殺 | 日柱の地支（配偶者宮） | 配偶者縁の課題。孤独・独立・パートナーシップの変容 |
| 年干中殺 | 年柱の天干（星の中殺） | 先天的な才能の変容。社会的な顔の特殊性 |
| 月干中殺 | 月柱の天干（星の中殺） | 職業・社会的立場に関わる星の変容 |

💡 日支中殺（配偶者宮の中殺）は特に重要視されます。結婚・パートナーシップに課題を持つ「孤独の宿命」として扱われることがありますが、必ずしも不幸ではなく、一人で完結する強さを持つとも解釈されます。


## 6.3 天中殺の時期の過ごし方

天中殺の年・月は「気」が乱れやすく、新しい行動・決断・契約を避けるべき時期とされます。ただし、「超天中殺（ちょうてんちゅうさつ）」の概念もあり、あえて動くことで吉になるケースもあります。


| 天中殺中に避けること | 新規事業の開始、重要な契約・結婚・引越し、大きな金銭の動き |
| --- | --- |
| 天中殺中に良いこと | 内省・学習・準備・精神修養・過去の整理 |
| 超天中殺 | 天中殺の2年間を完全にアクティブに動き切ることで、次の10年の運を底上げするという考え方 |
