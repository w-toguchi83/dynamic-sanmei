| 干支 | かんし | 十干と十二支の組み合わせ。年・月・日の「気」を表す |
| 日干 | にっかん | 日柱の天干。命式で最も重要で「自分自身」を表す |

| 蔵干 | ぞうかん | 十二支の内部に格納された十干 |

| 宿命中殺 | しゅくめいちゅうさつ | 命式の中の干支が天中殺に当たること |
| 陽占 | ようせん | 天干・十大主星を中心とした命式の読み方 |
| 陰占 | いんせん | 地支・十二大従星を中心とした命式の読み方 |
| 大運 | たいうん | 10年ごとに変化する長期的な運勢の流れ |
| 年運 | ねんうん | 1年ごとに変化する運勢 |
| 三才 | さんさい | 天（年干）・人（月干）・地（日干）の三位置 |
| 格局 | かくきょく | 命式全体の傾向・強さの格式 |
| 五行 | ごぎょう | 木・火・土・金・水の五つの基本エネルギー |

## 12.2 流派・参考文献について

算命学には複数の流派・解釈があります。Webサービス実装にあたっては、採用する流派・参考書を明示し、計算ロジックを統一することが重要です。


| 日本算命学の開祖 | 高尾義政（たかおよしまさ）氏 ※戦後に体系化 |
| --- | --- |
| 主要参考文献 | 「算命学大系」高尾義政著 ほか |
| 注意事項 | 流派によって蔵干・格局判定・大運の起算方法が異なる場合があります |
| 推奨アプローチ | システム設計時に採用流派を明示し、ロジックをドキュメント化する |

付録B　実装チェックリスト


## B.1 計算精度チェック

- 節入日時の精確なデータソースを確保しているか
- 日干支算出の基準日と計算式を文書化しているか
- タイムゾーン・日付変更の処理を明確にしているか
- 立春前後の年柱切り替えを正しく実装しているか
- 月干算出（五虎遁年法）を実装しているか
- 蔵干の取得ロジックが流派に対応しているか

## B.2 機能チェック

- 三柱（年・月・日）の干支が正確に算出されるか
- 十大主星が全ての干に対して正しく算出されるか
- 十二大従星が全ての地支に対して正しく算出されるか
- 天中殺の判定が正確か
- 宿命中殺の判定が正確か
- 大運の起算・進行方向（順行・逆行）が正しいか
- 合・冲・刑・害の判定機能があるか

## B.3 UI/UXチェック

- 命式盤が視覚的にわかりやすいか
- 難しい用語にルビ・解説があるか
- 計算根拠・採用流派が明示されているか
- 天中殺期間がカレンダーで確認できるか
- 鑑定結果のエクスポート機能があるか
