# 第11章　応用概念と発展的な技法


## 11.1 納音（なっちん）

六十干支を五行と音の観点から30組に分類したもの。各組には固有の名称と象意があります。

💡 納音は古代中国の音楽理論と干支を結びつけた概念で、算命学でも補助的に使用されます。


## 11.2 空亡（くうぼう）と天中殺の違い

四柱推命で使われる「空亡」と算命学の「天中殺」は同じ原理から生まれた概念ですが、算命学では天中殺と呼びます。六十干支の10干と12支のうち余る2支が「空亡（天中殺）支」となります。


## 11.3 二十八宿（にじゅうはちしゅく）

月の軌道を28に分けた星宿。日柱の算出に関連することがあります。算命学では補助的概念として扱われることがあります。


## 11.4 龍高星・玉堂星の「海外運」

龍高星（印綬・陽）は変化・海外・異文化との縁を示します。龍高星が天や人の位置にある人は海外との縁が強いとされます。


## 11.5 位相法（いそうほう）

命式の干支同士の「合・冲・刑・害」を系統的に分析する方法。大運・年運が命式に作用する際の変化を読む技法です。


## 11.6 流年法（りゅうねんほう）

大運・年運・月運を重ね合わせて「いつ何が起きやすいか」の具体的なタイミングを読む技法。


| 大運（10年） | 長期的なテーマ・方向性 |
| --- | --- |
| 年運（1年） | その年の主要な出来事・チャンス・課題 |
| 月運（1ヶ月） | 具体的な動きのタイミング |
| 日運（1日） | より細かい日次の流れ（上級者向け） |
