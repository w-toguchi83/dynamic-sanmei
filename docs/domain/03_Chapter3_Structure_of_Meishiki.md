# 第3章　命式（めいしき）の構成


## 3.1 三柱の概要

算命学では生年・生月・生日から「年柱・月柱・日柱」の三柱を立てます。各柱は「天干（てんかん）」と「地支（ちし）」の上下一対で構成されます。


| 柱 | 天干（上） | 地支（下） | 象意 |
| --- | --- | --- | --- |
| 年柱（ねんちゅう） | 年干 | 年支 | 先祖・社会・外面の運・0〜20歳頃 |
| 月柱（げっちゅう） | 月干 | 月支 | 両親・社会的立場・中年期（20〜40歳） |
| 日柱（にっちゅう） | 日干 | 日支 | 自分自身・配偶者・壮年期（40〜60歳） |

💡 日干（にっかん）は「自分そのもの」を表す最重要の干で、「日主（にちしゅ）」とも呼ばれます。全ての星はこの日干を基準に算出されます。


## 3.2 命式の起こし方（基本手順）


### 3.2.1 年柱の算出

年柱は生まれた年の干支を用います。ただし算命学では「節入り（せつにり）」を年の区切りとする場合と、元旦（1月1日）を区切りとする場合があります。一般的には「立春（2月4日頃）」を年の区切りとします。

⚠ 2月4日（立春）以前に生まれた場合、前年の年柱を使用します。これはシステム設計で最も注意が必要なポイントの一つです。


### 3.2.2 月柱の算出

月柱は生まれた月の干支。月支は固定ですが、月干は年干によって変化します。月の区切りは各月の「節入り日」（二十四節気の節）です。


| 月支 | 旧暦月 | 節入り（概算） | グレゴリオ月 |
| --- | --- | --- | --- |
| 寅 | 1月 | 立春（2/4頃） | 2月 |
| 卯 | 2月 | 啓蟄（3/6頃） | 3月 |
| 辰 | 3月 | 清明（4/5頃） | 4月 |
| 巳 | 4月 | 立夏（5/6頃） | 5月 |
| 午 | 5月 | 芒種（6/6頃） | 6月 |
| 未 | 6月 | 小暑（7/7頃） | 7月 |
| 申 | 7月 | 立秋（8/8頃） | 8月 |
| 酉 | 8月 | 白露（9/8頃） | 9月 |
| 戌 | 9月 | 寒露（10/8頃） | 10月 |
| 亥 | 10月 | 立冬（11/7頃） | 11月 |
| 子 | 11月 | 大雪（12/7頃） | 12月 |
| 丑 | 12月 | 小寒（1/6頃） | 翌1月 |


### 3.2.3 日柱の算出

日柱は生まれた日の干支。60日周期で繰り返します。基準日からの経過日数を60で割った余りから算出します。精確な計算にはユリウス通日（Julian Day Number）を用いる方法が一般的です。

💡 日干支の計算はアルゴリズムで実装可能です。基準日（例：1900年1月1日＝甲戌）からの日数差をmod 60することで干支インデックスを求めます。


## 3.3 後天運（こうてんうん）


### 3.3.1 大運（たいうん）

大運は10年ごとに変化する長期的な運勢の流れです。月柱を基準に、生まれた月の「節入り日」からの日数を3で割った値が最初の大運の年数（数え年）となります。


| 性別・年干の陽陰 | 大運の進み方 |
| --- | --- |
| 陽年干の男性・陰年干の女性 | 順行（月柱の次の干支から順に進む） |
| 陰年干の男性・陽年干の女性 | 逆行（月柱の前の干支から逆に進む） |


### 3.3.2 年運（ねんうん）・歳運（さいうん）

その年の干支（太歳）が命式に与える影響を「年運」または「歳運」と呼びます。毎年変化し、大運と組み合わせて読みます。


### 3.3.3 月運（げつうん）

その月の干支が命式に与える影響。より短期的な運勢の変化を読みます。大運・年運・月運を重ね合わせることで具体的なタイミングを絞り込みます。
