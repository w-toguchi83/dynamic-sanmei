# 付録B　実装チェックリスト

## B.1 計算精度チェック

- 節入日時の精確なデータソースを確保しているか
- 日干支算出の基準日と計算式を文書化しているか
- タイムゾーン・日付変更の処理を明確にしているか
- 立春前後の年柱切り替えを正しく実装しているか
- 月干算出（五虎遁年法）を実装しているか
- 蔵干の取得ロジックが流派に対応しているか
## B.2 機能チェック

- 三柱（年・月・日）の干支が正確に算出されるか
- 十大主星が全ての干に対して正しく算出されるか
- 十二大従星が全ての地支に対して正しく算出されるか
- 天中殺の判定が正確か
- 宿命中殺の判定が正確か
- 大運の起算・進行方向（順行・逆行）が正しいか
- 合・冲・刑・害の判定機能があるか
## B.3 UI/UXチェック

- 命式盤が視覚的にわかりやすいか
- 難しい用語にルビ・解説があるか
- 計算根拠・採用流派が明示されているか
- 天中殺期間がカレンダーで確認できるか
- 鑑定結果のエクスポート機能があるか
