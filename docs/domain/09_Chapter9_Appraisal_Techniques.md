# 第9章　鑑定技法と読み方


## 9.1 命式全体の読み方の手順

算命学の鑑定は以下の手順で行うのが基本です。

- 命式を起こす（年柱・月柱・日柱を確定）
- 日干を確認し、基本的な気質・五行の強さを把握
- 天中殺の種類を確認
- 宿命中殺の有無と位置を確認
- 十大主星の三才配置を読む（陽占）
- 十二大従星の三才配置を読む（陰占）
- 五行バランスを確認し、強い・弱い五行を把握
- 大運の流れを確認（10年ごとの運勢の大きな流れ）
- 年運・月運と命式・大運の関係を読む
- 総合的に統合して鑑定文をまとめる

## 9.2 相性鑑定（二人の命式の比較）

二人の命式を並べて比較することで、相性・関係性・課題を読みます。


| 相性の観点 | 内容 |
| --- | --- |
| 五行の補完 | お互いの弱い五行を補い合えるか |
| 天中殺の関係 | お互いの天中殺支が相手の命式にあるか |
| 主星の組み合わせ | お互いの主星が相生か相剋か |
| 宿命中殺の影響 | 一方の中殺が他方に影響を与えるか |
| 大運の同期 | 重要な転換期が重なるか |


## 9.3 特殊な命式パターン


### 9.3.1 貫索・石門が天にある命式

貫索星または石門星が年干（天の位置）にある場合、先天的な強い自我または社会性を示します。


### 9.3.2 日支中殺（孤独の命）

日支が天中殺に当たる場合、配偶者縁の課題・孤独の宿命を持つとされます。一方で強い精神的独立心と才能を持ちます。


### 9.3.3 月支・年支が同じ従星

二つの位置に同じ従星が来る場合、そのエネルギーが二重に強調されます。
