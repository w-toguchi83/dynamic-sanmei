# 第7章　陰占（いんせん）と陽占（ようせん）


## 7.1 陰占の概要

陰占は命式の「地支」を中心とした読み方で、より深層の宿命・潜在的なエネルギー・肉体的側面を表します。十二大従星が主役です。


| 陰占の要素 | 内容 |
| --- | --- |
| 地支の星 | 年支・月支・日支の十二大従星 |
| 三業（さんごう） | 年支・月支・日支の従星の組み合わせパターン |
| 天中殺 | 生まれ持つ天中殺支 |
| 宿命中殺 | 命式内の中殺の有無と位置 |
| 納音（なっちん） | 干支の組み合わせから算出される特殊なカテゴリ |


## 7.2 陽占の概要

陽占は命式の「天干」を中心とした読み方で、表面的な気質・行動パターン・社会的な顔を表します。十大主星が主役です。


| 陽占の要素 | 内容 |
| --- | --- |
| 天干の星 | 年干・月干・日干（日干自体）の十大主星 |
| 三才（さんさい） | 天・人・地の三位置における星の分析 |
| 五行バランス | 命式全体の五行の過不足 |
| 格局（かくきょく） | 命式の全体的な傾向・格式 |


## 7.3 三才論（さんさいろん）

三才とは「天・人・地」の三つの位置で、陽占における命式の読み方の基軸です。


| 三才の位置 | 対応 | 意味 |
| --- | --- | --- |
| 天（てん） | 年干の星 | 先祖・社会・外部からの影響・若年期 |
| 人（じん） | 月干の星 | 両親・職業・社会的立場・中年期 |
| 地（ち） | 日干（日主） | 自分自身・意志・晩年期 |


### 7.3.1 三才のバランスと人生パターン

三才に配置された星の種類と組み合わせから、その人の人生の流れや強みが読めます。例えば「天剋地冲（てんこくちちゅう）」の命式は激しい変動を持つとされます。


## 7.4 格局（かくきょく）

格局とは命式全体の傾向・力関係を一言で表す概念です。日干の強さ（身強・身弱）と五行の分布から判断します。


| 格局の種類 | 説明 |
| --- | --- |
| 身強（みつよ）格 | 日干が強い状態。自己主張が強く、独立心旺盛 |
| 身弱（みよわ）格 | 日干が弱い状態。周囲の影響を受けやすく、協調性がある |
| 従旺格（じゅうおうかく） | 日干が極めて強く、剋する星がない特殊格 |
| 従弱格（じゅうじゃくかく） | 日干が極めて弱く、完全に周囲に従う特殊格 |
| 化気格（かきかく） | 日干が特定の合化をして五行が変化する特殊格 |
