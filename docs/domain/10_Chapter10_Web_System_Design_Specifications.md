## 10.2 算出アルゴリズムの実装要点


### 10.2.1 日干支の算出

日干支の計算は以下の公式で実装します。

// 基準日: 1900年1月1日 = 甲戌（インデックス: 10）

function getDailyKanshi(date) {

  const base = new Date(1900, 0, 1); // 甲戌 = index 10

  const diffDays = Math.floor((date - base) / 86400000);

  const idx = ((10 + diffDays) % 60 + 60) % 60;

  const kanIdx = idx % 10; // 十干インデックス（0-9）

  const shiIdx = idx % 12; // 十二支インデックス（0-11）

  return { kan: JIKKAN[kanIdx], shi: JUNISHI[shiIdx] };

}

⚠ 日付変更線・タイムゾーンの処理に注意。深夜0時以降を翌日とするか、子の刻（23:00〜1:00）を前日として扱うかで結果が変わります。


### 10.2.2 節入日の取得

月柱・年柱の正確な算出のために「節入日時」が必要です。節入日は毎年変わるため、動的に計算するか、事前計算したデータベースを使用します。


| 推奨実装 | 太陽黄経を基に節入日時を計算するライブラリを使用（例：Swiss Ephemeris, astronomical-algorithms） |
| --- | --- |
| 代替実装 | 1900年〜2100年の節入日時をJSON/DBに事前格納 |
| 精度要件 | 時分秒まで精確に（節入当日の生まれは特に重要） |


### 10.2.3 月干の算出（五虎遁年法）

月干は年干によって決まります。「五虎遁年法（ごことんねんほう）」と呼ばれる規則を使います。


| 年干 | 1月（寅）の月干 | 規則 |
| --- | --- | --- |
| 甲・己 | 丙 | 甲己年の1月は丙寅から始まる |
| 乙・庚 | 戊 | 乙庚年の1月は戊寅から始まる |
| 丙・辛 | 庚 | 丙辛年の1月は庚寅から始まる |
| 丁・壬 | 壬 | 丁壬年の1月は壬寅から始まる |
| 戊・癸 | 甲 | 戊癸年の1月は甲寅から始まる |


## 10.3 機能要件の整理


### 10.3.1 コア機能

- 命式起こし機能（生年月日から三柱の干支を算出）
- 十大主星算出機能
- 十二大従星算出機能
- 天中殺判定機能
- 宿命中殺判定機能
- 大運算出機能（10年ごとの運勢サイクル）

- 命式の総合鑑定文生成（AI連携含む）

- 年間運勢カレンダー生成
- 天中殺カレンダー表示
- 五行バランスレーダーチャート

### 10.3.3 学習・教育機能（プロ・教育者向け）

- 命式の各要素の詳細解説表示

- 星の意味・算出根拠の可視化
- 流派別設定の切り替え機能
